# General Knowledge Quiz (Python)

## Description 
A simple command-line General Knowledge Quiz built using Python.
This project tests the user's knowledge through questions while demonstration basic Python programming Concepts.

## Features
-Multiple general knowledge questions
-User input for answering questions
-Score calculation
-Displays final quiz result 

## Technologies Used
-Python

## How to Run
1. Make sure Python is installed.
2. Download or clone this repository.
3. Run the project.

## Concepts Used
-Loops 
-Condtitional statements('if-else')
-Functions
-User input('input()')
